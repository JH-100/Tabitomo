# Tabitomo (タビトモ / 旅友)

A Japanese conversation learning app for travelers heading to Japan.

## Tech Stack

- Expo SDK 54 + React Native 0.81
- TypeScript
- Expo Router v6
- Zustand + AsyncStorage
- Supabase
- React Native Reanimated v4

## Getting Started

```bash
npm install
npx expo start
```
