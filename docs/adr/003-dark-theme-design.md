# ADR-003: 다크 테마 전용 디자인

## 상태

승인됨

## 맥락

앱 테마 전략 결정 필요. 라이트/다크 토글 vs 다크 전용.

## 결정

다크 테마 전용 (딥스페이스 그라디언트).

## 이유

- "여행" 테마와 밤하늘/우주 분위기 일치
- 글래스모피즘 UI가 어두운 배경에서 효과적
- 초기 개발 단계에서 테마 토글 구현 부담 감소
- 타겟 사용자(여행 앱) 사용 시간대가 다양

## 결과

- Colors.ts에 다크 팔레트만 정의
- ScreenContainer가 그라디언트 배경 일관 적용
- 추후 라이트 모드 추가 시 Colors.ts 확장 필요
