# Features

## 1. 온보딩 플로우

### 1.1 웰컴 캐러셀

- 3페이지 스와이프 소개
- 페이지: 일본어 학습 소개 → 여행 동반자 컨셉 → 학습 시작 유도
- Skip/Next 버튼, 페이지 인디케이터
- Reanimated 진입 애니메이션

### 1.2 여권 생성

- 입력 필드: 이름, 생년월일, 성별
- 실시간 유효성 검증 (canSubmit)
- PassportScene: 3D 여권 시각화 (플립 애니메이션)
- 승인 도장 애니메이션
- 완료 시 hasCompletedOnboarding = true

## 2. 홈 대시보드

- 환영 인사 (유저 이름 / 일본어 인사)
- 스트릭 카운터 (연속 학습 일수)
- 히어로 카드: 아바타 + 현재 위치 + 레벨
- 퀵 액션: 다음 레슨, 퀴즈, 매칭 (레슨 1개 이상 완료 필요)
- 여행 지도: 전체 진행도 시각화

## 3. 레슨 시스템

### 3.1 레슨 목록

- 히라가나 섹션 (10레슨) + 카타카나 섹션 (10레슨)
- 진행률 바 (완료/전체)
- 선형 잠금 해제: 이전 레슨 완료 필수
- 카타카나는 히라가나 전체 완료 후 해금

### 3.2 레슨 상세 (플래시카드)

- 문자 1개씩 제시 → 4지선다 (로마지 맞추기)
- 정답/오답 시각 피드백 (색상 + 아이콘)
- 오답 시 힌트(예시 단어) 표시
- 진행률 바 + 문자 카운터
- 완료 화면: 정답률, 획득 XP 표시

## 4. 연습 모드

### 4.1 퀴즈 (4지선다)

- 완료한 레슨 범위에서 10문제 출제
- 방향 랜덤: 히라가나→로마지 / 로마지→히라가나
- 결과: 점수, 정답/오답 수, XP

### 4.2 입력 (로마지 타이핑)

- 히라가나 표시 → 로마지 직접 입력
- 대소문자 무시 비교
- 키보드 Return으로 제출/다음

### 4.3 매칭 (카드 게임)

- 5쌍 10장 카드 뒤집기
- 히라가나↔로마지 짝 맞추기
- 시도 횟수 기반 효율 점수
- 스프링 애니메이션 피드백

### 4.4 필기 (드로잉)

- 학습한 문자 중 선택
- SVG 캔버스에 자유 필기
- 참조 문자 워터마크
- 지우기/이전/다음 제어
- 채점 기능 없음 (자유 연습)

## 5. 프로필

- 아바타 표시 (글로우 효과)
- 기본 정보: 이름, 생년월일
- 통계: 레벨, 스트릭, XP
- 리셋 버튼 (전체 초기화)

## 6. 게이미피케이션

### 레벨 시스템

| 레벨 | 장소        | 조건          |
| ---- | ----------- | ------------- |
| 0    | 여권 발급   | 온보딩 완료   |
| 1    | 항공권 구매 | 히라가나 전체 |
| 2    | 짐 싸기     | 카타카나 전체 |
| 3~8  | 공항~오사카 | 미구현        |

### XP 시스템

- 레슨 완료: 20~25 XP (고정)
- 연습 완료: 최대 10 XP (정답 비율 기반)
- 누적 표시, 소비 메커니즘 없음

### 여행 지도

- 9단계 시각화 (좌우 교차 배치)
- 현재 레벨에 아바타 표시
- 잠금/완료/현재 상태 구분
