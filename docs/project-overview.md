# Project Overview

## Tabitomo (タビトモ / 旅友)

**한 줄 요약:** 일본 여행자를 위한 게이미피케이션 일본어 학습 앱

## 프로젝트 목표

1. 일본 여행 준비 과정에서 필수 일본어(히라가나/카타카나/기초 회화)를 학습
2. 여행 여정 컨셉으로 학습 동기부여 (도시 해금 시스템)
3. iOS + Android + Web 크로스 플랫폼 배포
4. 오프라인 우선, 추후 클라우드 동기화

## 타겟 사용자

- 일본 여행을 준비하는 한국인
- 일본어 초보자 (문자부터 학습)
- 짧은 시간 집중 학습 선호

## 핵심 가치

| 가치          | 설명                                  |
| ------------- | ------------------------------------- |
| 여행 내러티브 | 학습 = 여행 준비, 레벨 = 도시 도착    |
| 단계적 해금   | 이전 단계 완료 → 다음 단계 잠금 해제  |
| 다양한 연습   | 퀴즈, 입력, 매칭, 필기 4가지 모드     |
| 시각적 보상   | 여권, 아바타, 여행 지도로 진행 시각화 |

## 현재 상태

- **버전:** 1.0.0 (프리알파)
- **커밋:** 2개 (초기 개발 단계)
- **컨텐츠:** 히라가나 10레슨 + 카타카나 10레슨
- **레벨:** Lv.0~2 구현 (Lv.3+ 미구현)
- **백엔드:** Supabase 클라이언트만 설치, 미연동
- **테스트:** 없음
- **배포:** Expo Go 개발 환경만

## 기술 스택 요약

- Expo SDK 54 / React Native 0.81 / React 19
- TypeScript (strict) / Expo Router v6
- Zustand 5 + AsyncStorage
- React Native Reanimated v4
- Supabase (계획중)
